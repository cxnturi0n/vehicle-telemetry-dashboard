## Try it

1. Clone the repository:
```
git clone https://github.com/cxnturi0n/vehicle-telemetry-dashboard.git
```
2. Navigate to the root directory:
```
cd vehicle-telemetry-dashboard
```
3. Run
```
docker compose up
```
